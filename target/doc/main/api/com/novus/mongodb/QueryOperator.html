trait QueryOperator
[source: com/novus/mongodb/QueryOperator.scala]
sealed trait QueryOperator
extends AnyRef
Base trait for QueryOperators, children are required to define a value for field, which is a String and refers to the left-hand of the Query (e.g. in Mongo: {"foo": {"$ne": "bar"}} "foo" is the field.

Method Summary
protected def op (op : java.lang.String, target : Any) : (java.lang.String, com.mongodb.DBObject)
Base method for children to call to convert an operator call into a Mongo DBObject. e.g. "foo" $ne "bar" will convert to {"foo": {"$ne": "bar"}} Optionally, if dbObj, being Some(DBObject) is defined, the op() method will nest the target value and operator inside the existing dbObj - this is useful for things like mixing $lte and $gte WARNING: This does NOT check that target is a serializable type. That is, for the moment, your own problem.
