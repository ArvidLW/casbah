mongo-scala-wrappers 0.1.1 API

This document is the API specification for mongo-scala-wrappers 0.1.1 API

Class Summary
class MapReduceCommand extends AnyRef
Wrapper class for invoking MongoDB mapReduces. The Java driver doesn't provide support for many of the possible options in the latest versions of MongoDB, so this wrapper class is used in it's place, and passed directly to a db.runCommand call.
class MapReduceResult (resultObj : com.mongodb.DBObject, implicit db : ScalaMongoDB) extends scala.Iterator[com.mongodb.DBObject] with Logging
Wrapper for MongoDB MapReduceResults, implementing iterator to allow direct iterator over the result set.
Object Summary
object MapReduceCommand extends AnyRef
Wrapper Object to provide apply methods for the MapReduceCommand class.
